# good-old-snake
Basic snake game in Java\
Using java.awt libraries

## How to play:
Arrow keys to control the snake\
P - pause\
R - restart\
E - enable/disable grid\
Esc - quit
### M - main menu: 
Enter - select\
Up and Down arrows - choose\
\
Snake can go through the walls!
